ScoreFlash - PlayMaker Addon
V4.6.1, 2014-12-27

This addon provides integration between ScoreFlash and
PlayMaker. To be able to use it, you need to have
both packages installed in your project.

You can get Score Flash on the Unity Asset Store:
https://www.assetstore.unity3d.com/en/#!/content/4476

You can also get PlayMaker on the Unity Asset Store:
https://www.assetstore.unity3d.com/en/#!/content/368
